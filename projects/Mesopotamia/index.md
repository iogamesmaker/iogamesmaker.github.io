# Mesopotamia
In the Netherlands some students have to do a finals project before they graduate high school.<br>
For our finals project, we made a little videogame!<br>
My friend did 99% of the art, I did the code bit. Raw C++, only external library is some Perlin noise library and SDL2. Rawdogged it.<br>
If you're interested in playing it, be warned, it's still in development. Gameplay is (in my opinion) pretty fun, but a little unbalanced. It's on the harder side of things.<br><br>
