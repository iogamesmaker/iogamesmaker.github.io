# GPU Raymarcher
Made my own little custom shadertoy-esque thing a while back with help from [LearnOpenGL](https://learnopengl.com)<br>
This and the GPU black hole renderer are both made with that as the basis.<br><br>
It's an infinite terrain which you can explore. Performance is pretty alright, runs fine on my GTX 970.<br><br>
I definitely want to do more raymarching stuff like this in the future. It's been 2 years since I wrote this program, on April 14th, 2024. That's crazy<br><br>
