Raymarched Landscape

I recently dived into the surprisingly deep voxel rabbithole, and needed to start somewhere, so I made a meh raytracer. I got kind-of offtrack yesterday evening and converted it to a raymarcher and added a simple procedural world, which this cool webpage is about.

You can find the source code here.

Cool features:


- Fully procedural raymarched landscape + raytracing mode
- Broken terrain SDF
- downscaling mode for more FPS!!!!!!!!
- Epic hacked-in reflections
- Amazing borders between biomes
- Yeah! BIOMES!!!! (desert thing, mountains, oceans, default/plains)


- very cool controls
