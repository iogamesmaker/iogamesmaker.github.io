⯆ Instructions ---------- JS userscript

1: Install Tampermonkey

2: Enable your browser's developer mode. Tutorial

3: Click on the Tampermonkey extension icon, and click "create new script".

4: Paste the script you want into the text editor, and press CTRL + S.

5: For some scripts that listen to commands, I have my username hardcoded in the script. Replace that with your username.

6: Reload your drednot.io tab, and try it out. Further instructions might be inside of the script.

⯆ Instructions ---------- Python script

This is more tricky to install, might need some of my help. DM @iogamesplayer if you get stuck.

1: Install the Python 3.11 (or newer idk when you read this) package from the Microsoft Store.

2: Download the script into your Downloads folder as "log.py".

3: Open CMD, and type "cd Downloads".

4: Run "python3 log.py" - it should return some errors. If the error is "Python3 is not recognized as a command", I don't really know how to fix that. Try installing the package from python.org AND MAKE SURE YOU CHECK THE ADD PYTHON TO $PATH CHECKBOX!!!

5: All errors should be some imports missing, like "requests". Type "pip install XYZ" to install these

6: Repeat that for every missing package, until it finally works.

7: Wait like an hour for all the data to download. Read the help menu or something. It's ~1.5GB and should only download once.
